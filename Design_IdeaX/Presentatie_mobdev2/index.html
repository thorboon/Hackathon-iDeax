User scenario's

1e Scenario

Als gebruiker moet ik bij het registreren twee maal mijn wachtwoord ingeven. Ik geef de tweede maal een ander wachtwoord in dan het eerste wachtwoord. Ik krijg hierdoor een melding dat beide ww’en niet overeen komen en de registreer knop kan ik niet indrukken totdat ik twee overeenkomende ww’en intyp.

User scenario's

2e Scenario

Als geregistreerde gebruiker zou ik graag een nieuwe post maken. Hiervoor klik ik onderaan het scherm op het grote plusteken. Ik zal worden doorgewezen naar een pagina waar ik een gewenste foto moet selecteren en vervolgens op "next" moet klikken. Daar zal ik de titel, categorie en beschrijving invullen. Waarna ik op de postknop kan drukken voor publicatie.

User scenario's

3e Scenario

Als geregistreerde gebruiker zou ik graag mijn persoonlijk profiel zien. Dit kan doormiddel van het klikken op het profielicoontje rechtsboven in de navigatiebar. Hierbij zal ik mijn monthlysavings, een knop naar mijn gelikete posts, favoriete posts en eigen posts zien.
